# MkDocs

Markdown 的项目文档。

---

MkDocs 是一个 **快速**，**简单** 和 **华丽的** 静态站点生成器，用于构建项目文档。
文档源文件用 Markdown 编写，并使用单个 YAML 配置文件进行配置。
首先阅读[入门教程]，然后查看[用户指南]以获得更多信息。

[入门教程]: getting-started.md
[用户指南]: user-guide/README.md
